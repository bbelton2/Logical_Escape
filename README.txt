Game Name: The Logical Escape

How to Play:
Left arrow key moves you left.
Right arrow key moves you right.
Space is jump.
Up arrow key is used to interact with objects, such as picking them up or flipping switches.
Can only hold one object at a time
Down arrow key is used to drop an object you are holding
Each level has a door at the end, which promps a puzzle.
Use the mouse to solve the puzzles.
